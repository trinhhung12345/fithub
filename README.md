# fithub

A new Flutter project built with MVVM architecture.

## Getting Started

This project is a starting point for a Flutter application using the MVVM (Model-View-ViewModel) design pattern.

## Project Structure

- **lib/models** - Data models and entities
- **lib/views** - UI screens and widgets
- **lib/viewmodels** - Business logic and state management
- **lib/services** - API calls and external services
- **lib/repositories** - Data access layer
- **lib/utils** - Helper functions and constants
